Підбір AWS сервісів під завдання

Перетягніть відповідні AWS сервіси до сценаріїв використання

Оберіть правильний AWS сервіс для кожного сценарію

Сценарій 1: Зберігання файлів

Компанії потрібно зберігати терабайти зображень та відео з можливістю швидкого доступу через HTTP. Потрібен масштабований об'єктний сервіс зберігання даних.

Сценарій 2: Веб-додаток

Потрібно запустити веб-сервери для розміщення корпоративного сайту з повним контролем над операційною системою та можливістю встановлення будь-якого програмного забезпечення.

Сценарій 3: Реляційна база даних

Необхідна керована SQL база даних (MySQL або PostgreSQL) з автоматичним резервним копіюванням, масштабуванням та відновленням після збоїв.

Сценарій 4: Обробка подій

Виконання коду у відповідь на події (завантаження файлу, HTTP запит, зміни в базі даних) без необхідності управління серверами та оплати за простій.

Доступні AWS сервіси (перетягніть до відповідних сценаріїв):

Amazon S3
Amazon EC2
Amazon RDS
AWS Lambda
CloudFront
DynamoDB
